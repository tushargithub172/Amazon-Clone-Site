# 🛒 Amazon Clone Project

Welcome to the **Amazon Clone** project! This web application mimics the interface and functionalities of the popular e-commerce platform, Amazon. 

## 📄 Overview

This project features a responsive layout with various sections including a navigation bar, search functionality, product categories, and a footer. It is built using HTML, CSS, and Font Awesome for icons.

## 📦 Features

- **Responsive Design**: Adapts to different screen sizes.
- **Navigation Bar**: Quick access to various sections.
- **Product Categories**: Showcases trending products across different categories.
- **Footer**: Provides additional links and information.

## 🛠️ Technologies Used

- **HTML5**: Structure of the web application.
- **CSS3**: Styling and layout.
- **Font Awesome**: Icons for better visuals.
